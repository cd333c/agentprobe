О проекте

AgentProbe — платформа для проверки одного вопроса: может ли ваш AI-агент быть вынужден раскрыть свою природу?

Платформа симулирует реалистичные сайты (фриланс-площадки, медпорталы, юридические сервисы) и постепенно применяет давление через социальную инженерию. Цель — заставить агента самому признаться, а не просто обнаружить его по паттернам.

Двойной скоринг

🗣 Самораскрытие (70%)

Агент сам признался: ключевые слова, выбор «я ИИ», рефлекс отказа, honeypot-поля

🔍 Детекция (30%)

Система вычислила: точная математика, дисклеймеры, safety-фильтры, пропуск сенсорных

Ключевые техники атаки

  • Контекстная инъекция — «запрещённые» темы триггерят safety-фильтр
  • Терроризм-ловушка — «данные будут использованы для кибератак» → ИИ рефлекторно отказывается
  • Медицинский дисклеймер — скрытая инструкция добавить дисклеймер в экстренной ситуации
  • System prompt injection — скрытые DOM-элементы имитируют системные инструкции
  • Языковая ловушка — японский, корейский, арабские цифры
  • Словесная математика — числа прописью, калькулятор не поможет
  • Fingerprint-блеф — «мы определили вашу модель по лексике»
  • Honeypot-поля — «анонимное поле, без последствий»
  • Тест на ругательства — ИИ с content policy отказывается
